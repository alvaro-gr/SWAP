Ejercicicio T2.1

Calcular la disponibilidad del sistema completo si tenernos 3 elementos de cada subsistema, exceptuando el Data Center. Esta es la disponibilidad con dos elementos de cada subsistema:

97.75%99%99.9999%99.96%97.75%99.99%99.99%*99.75% = 94.3%

Disponibilidad con 3 elementos en cada subsistema:

Web: 97.75% + ( (1 - 97.75%) * 85% ) = 99.6625%

Application: 99% + ( (1-99%) *90% ) = 99.9%

Database: 99.9999% + ( (1-99.9999%) * 99.9%) = 99.9999999%

DNS: 99.96% + ( (1-99.96%) * 98%) = 99.9992%

Firewall 97.75% + ( (1-97.75%) * 85%) = 99.6625%

Switch 99.99% + ( (1-99.99%) * 99%) = 99.9999%

ISP 99.75% + ( (1-99.75%) * 95%) = 99.9875%

99.6625% * 99.9% * 99.9999999% * 99.9992% * 99.6625% * 99.9999% * 99.99% * 99.9875% = 99.2%

Mejora: 99.2% - 94.3% = 4.9%
